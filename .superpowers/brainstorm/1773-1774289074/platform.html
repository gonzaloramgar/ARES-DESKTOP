¿En qué plataforma construimos la app?

Esto determina qué tan futurista y transparente puede ser la UI

C# WPF

C# WPF

Máximo control sobre transparencias, blur, animaciones y efectos visuales nativos de Windows. Ideal para apps tipo HUD. Ya tienes app.cs en el proyecto.

Transparencia real • Blur de ventana • Animaciones fluidas • GPU rendering

Python Qt

Python + PyQt6 / PySide6

Cross-platform, transparencias funcionales, fácil de iterar. Efectos visuales modernos con menos boilerplate que WPF.

Transparencia • Multiplataforma • Fácil prototipado • OpenGL disponible

Electron

Electron (HTML/CSS/JS)

Libertad total de diseño con tecnologías web. CSS glassmorphism, animaciones CSS/canvas, Three.js para 3D. La UI más flexible posible.

Glassmorphism • Three.js / WebGL • Máxima libertad visual • Ventana sin bordes

Mi recomendación: WPF si quieres algo nativo y potente, Electron si quieres máxima libertad visual
